  <action>
1. Create `src/lib/stores/project-store.ts`:
   - Import `create` from `zustand`, `persist` and `createJSONStorage` from `zustand/middleware`
   - Import ProjectMeta, Step from `@/lib/types/project`
   - Import deleteProjectData from `@/lib/stores/project-persistence`
   - Define `ProjectListState` interface:
     ```
     projects: ProjectMeta[]
     activeProjectId: string | null
     createProject: () => string          // creates new ProjectMeta with crypto.randomUUID(), adds to list, returns id
     deleteProject: (id: string) => void  // removes from list, calls deleteProjectData(id), if deleted was active set activeProjectId to null
     setActiveProject: (id: string | null) => void  // sets activeProjectId
     updateProjectMeta: (id: string, meta: Partial<Pick<ProjectMeta, "horseName" | "levelId" | "levelDisplayName" | "currentStep">>) => void  // updates matching project's fields + updatedAt
     ```
   - Use `create<ProjectListState>()(persist(...))` with:
     - name: `"kur-projects"`
     - version: 1
     - storage: `createJSONStorage(() => localStorage)`
   - `createProject` action: generates id via `crypto.randomUUID()`, creates ProjectMeta with empty horseName, empty levelId/levelDisplayName, currentStep "level", createdAt/updatedAt as `new Date().toISOString()`, adds to projects array, sets as activeProjectId, returns id
   - `deleteProject` action: filters out project from list, calls `deleteProjectData(id)`, sets `activeProjectId: null` if the deleted project was active
   - `updateProjectMeta` action: maps over projects, spreads partial meta + sets `updatedAt: new Date().toISOString()` on matching id
   - Export as `useProjectStore`

2. Create `src/lib/stores/wizard-store.ts`:
   - Import `create` from `zustand`
   - Import ProjectData, DEFAULT_PROJECT_DATA, Step from `@/lib/types/project`
   - Import saveProjectData, loadProjectData from `@/lib/stores/project-persistence`
   - Import KurLevel from `@/data/kur-levels`, StrengthRating from `@/data/strength-options`, ArenaPath from `@/components/ArenaCanvas`
   - Define `WizardState` interface extending ProjectData with actions:
     ```
     setLevel: (level: KurLevel) => void         // sets selectedLevel, clears exerciseRatings and programOrder
     setHorseName: (name: string) => void
     setTemperament: (t: "calm" | "neutral" | "energetic") => void
     setExerciseRating: (id: number, rating: StrengthRating) => void
     setProgramOrder: (order: number[]) => void
     setArenaPaths: (paths: ArenaPath[]) => void
     setStep: (step: Step) => void
     loadProject: (id: string) => void           // calls loadProjectData(id), sets all fields from result or DEFAULT_PROJECT_DATA
     saveCurrentProject: (id: string) => void    // extracts data fields from state, calls saveProjectData(id, data)
     resetToDefaults: () => void                 // sets all data fields to DEFAULT_PROJECT_DATA values
     ```
   - This is a NON-persisted zustand store (no persist middleware). Persistence is manual via loadProject/saveCurrentProject.
   - `loadProject`: calls `loadProjectData(id)`, if result is non-null spreads result fields into state, else spreads DEFAULT_PROJECT_DATA
   - `saveCurrentProject`: extracts `{ selectedLevel, horseName, temperament, exerciseRatings, programOrder, arenaPaths, currentStep }` from `get()` and calls `saveProjectData(id, data)`
   - Export as `useWizardStore`
  </action>
