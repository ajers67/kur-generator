    **Hook signature:**
    ```typescript
    export function useAnimationPlayer(timeline: AnimationSegment[]): {
      marker: MarkerState | null;
      playing: boolean;
      currentTime: number;
      totalDuration: number;
      speed: number;
      play: () => void;
      pause: () => void;
      seek: (time: number) => void;
      setSpeed: (speed: number) => void;
    }
    ```

    **Implementation:**
    - Per D-15: Internal state: `{ currentTime: number, playing: boolean, speed: number }`
    - Per D-17: When `playing === true`, run a `requestAnimationFrame` loop that:
      1. Calculates deltaTime from last frame (using `useRef` for lastTimestamp)
      2. Advances `currentTime += deltaTime * speed`
      3. Clamps currentTime to [0, totalDuration]
      4. When currentTime >= totalDuration, auto-pause and reset to 0
    - Compute `marker` from `getPositionAtTime(timeline, currentTime)` — use `useMemo` keyed on currentTime + timeline
    - `play()`: sets playing = true
    - `pause()`: sets playing = false
    - `seek(time)`: sets currentTime, updates marker immediately
    - `setSpeed(s)`: updates speed multiplier (valid: 0.5, 1, 1.5, 2 per D-18/Phase 10 compatibility)
    - Per D-18: API designed for Phase 10 to call `seek()` from audio currentTime

    **Important:** Use `useRef` for the rAF ID and lastTimestamp to avoid stale closure issues. Clean up rAF on unmount and when `playing` changes to false. Use `useCallback` for all returned functions per project conventions.
